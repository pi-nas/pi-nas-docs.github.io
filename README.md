# Pi-NAS Documentation

Raspberry Pi Network Attached Storage Documentation.

## Development & Building

Change to `src` directory first.

### MkDocs - Start live-reloading docs server:
```
mkdocs serve
```

### Build static site:
```
mkdocs build 
```

### Move site to root:
```
mv site/* ../
```

### Remove old directory:
```
rm -rf site
```
